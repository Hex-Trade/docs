# Getting Started

Follow these steps to start using Hextrade.

## 1. Create an account

Sign up for free (while in Beta).

**[Create your Hextrade account](https://dash.hextrade.io/en/auth/register)**

Use your email or sign up with Google.

## 2. Log in

1. Go to the [Hextrade dashboard](https://dash.hextrade.io).
2. Enter your credentials or use **Login with Google**.
3. After login you’re taken to the **Accounts** page.

## 3. Get around the dashboard

The **sidebar** (left) is your main navigation:

| Item | What it’s for |
| ---- | ---------------- |
| **Accounts** | Your connected trading accounts, portfolio value, and trades. |
| **Algos** | Trading algorithms — browse and use them. |
| **Copy Trade** | Copy-trade groups (master + followers). |
| **Analytics** | Performance and analytics. |
| **Economic calendar** | Economic events and calendar. |
| **Subscriptions** | Your algorithm subscriptions. |

The **header** (top) has search, notifications, and your profile (including profile picture and settings).
