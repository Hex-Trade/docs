# Getting Started

Follow these steps to start using Hextrade.

---

## 1. Create an account

Sign up for free (while in Beta).

<Card title="Create your Hextrade account" icon="user-plus" href="https://dash.hextrade.io/signup">
Use your email or sign up with Google.
</Card>

---

## 2. Log in

1. Go to the [Hextrade dashboard](https://dash.hextrade.io).
2. Enter your credentials or use **Login with Google**.
3. After login you’re taken to the **Accounts** page.

---

## 3. Connect a broker account

To trade, run algorithms, or copy trade, you need at least one connected broker account.

1. On the **Accounts** page, use **Add account** (or the equivalent control).
2. Choose your broker (e.g. **Tradier**, **DxTrade**, **Alpaca**, **CQG**, **ProjectX**, **Tradovate**).
3. Complete the broker’s sign-in or API key flow.
4. The account appears in your Accounts table with its platform and status.

For the full list and broker-specific notes, see [Supported brokers & accounts](/supported-brokers).

---

## 4. Get around the dashboard

The **sidebar** (left) is your main navigation:

| Item | What it’s for |
| ---- | ---------------- |
| **Accounts** | All connected broker accounts, portfolio value, and trades. |
| **Algos** | Trading algorithms — browse, subscribe, and run them. |
| **Copy Trade** | Copy-trade groups (master + followers). |
| **Analytics** | Performance and analytics. |
| **Economic calendar** | Economic events and calendar. |
| **Subscriptions** | Your algorithm subscriptions and plan limits. |
| **Trade** | Full-screen trading with charts and order entry. |
| **Chart settings** | Customize chart appearance. |
| **Settings** | Profile, theme, and app preferences. |

The **header** (top) has search, notifications, and your profile (picture and settings).
