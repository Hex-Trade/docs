# Import CSV

Use `Import CSV` to load portfolio allocations quickly instead of entering each algorithm manually.

> [GIF placeholder: Importing a portfolio CSV]

---

## Typical use cases

- Restore a saved research allocation.
- Share reproducible setups across team members.
- Move between test and production portfolios faster.

---

## Import workflow

1. Open `Portfolio` and click `Import CSV`.
2. Upload a correctly formatted CSV.
3. Validate the parsed algorithm rows and quantities.
4. Apply import and review tabs before deployment.

> [Image placeholder: Import CSV modal with parsed rows]

---

## Troubleshooting

- Invalid rows: correct algorithm IDs and quantity fields.
- Missing algorithms: verify availability on your account plan.
- Unexpected sizing: run a quick review in [Portfolio Analytics](/portfolio/analytics) and [Trades](/portfolio/trades) before deploying.
