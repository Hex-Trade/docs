# Deploy

`Deploy` applies your selected portfolio configuration to a target account so enabled algorithms can run live.

> [GIF placeholder: Deploying portfolio to account]

---

## Deployment requirements

- At least one enabled algorithm in the builder.
- A connected supported account (see [Supported Brokers](/supported-brokers)).
- Required subscription level for deployment features.

---

## Deployment workflow

1. Validate portfolio in all analysis tabs.
2. Click `Deploy`.
3. Select target platform/account.
4. Confirm deployment and monitor result statuses.

> [Image placeholder: Deploy modal with per-algorithm status]

---

## Post-deploy checklist

- Confirm subscriptions are active for selected algorithms.
- Monitor live behavior in [Accounts](/accounts) and [Analytics](/analytics).
- Revisit allocation if live behavior deviates from expectations.

<Warning>
Live deployment executes real strategy risk. Start with conservative sizing and scale only after stable validation.
</Warning>
