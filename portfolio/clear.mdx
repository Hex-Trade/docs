# Clear

Use `Clear` to remove all algorithms from the current portfolio builder session.

> [GIF placeholder: Clearing portfolio selection]

---

## When to use Clear

- Start fresh after testing a setup.
- Reset the builder before importing a new CSV.
- Remove stale selections before a final allocation run.

---

## Safe usage checklist

1. Confirm you no longer need the current configuration.
2. Use `Share` or screenshots for recordkeeping if needed.
3. Click `Clear` and confirm.
4. Rebuild or import your next target allocation.

> [Image placeholder: Clear confirmation modal]

---

## Common mistake to avoid

- Clearing before saving a promising setup. Use [Share](/portfolio/share) first if you might need to restore it.
