# Profile

The `Profile Information` and `Profile Picture` sections let you keep your account identity accurate and operational.

> [GIF placeholder: Editing profile and uploading avatar]

---

## Profile fields

- First name and last name.
- Email address (read-only).
- Phone number with country code.
- TradingView username (plan-dependent).
- Discord username (without `@`).

---

## Profile picture

- Upload supported image formats (`JPG`, `PNG`, `GIF`, `WebP`).
- Keep file size within platform limits.
- Use a square image for best rendering quality.

> [Image placeholder: Profile information and avatar panels]

---

## Best practices

- Keep phone number and Discord username current for support workflows.
- Use a recognizable avatar for team and community interactions.
- Save changes after each field update group.
