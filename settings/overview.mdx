# Settings Overview

The **Settings** area is your control center for account identity, security posture, billing, alerts, visual preferences, and API access.

> [GIF placeholder: Settings page full navigation tour]

---

## Settings sections

- [Profile](/settings/profile): Personal details, avatar, TradingView, and Discord username.
- [Security](/settings/security): Password updates and two-factor authentication (2FA).
- [Billing](/settings/billing): Plan status, billing period, and promo code flow.
- [Notifications](/settings/notifications): Channel and alert-type preferences.
- [Appearance](/settings/appearance): Theme and motion settings.
- [API Token](/settings/api-token): Programmatic trade feed access.

---

## Recommended setup order

1. Complete profile details.
2. Enable 2FA before connecting live accounts.
3. Confirm billing plan and limits.
4. Configure notifications for trade and connection events.
5. Generate API token only if needed for integrations.
