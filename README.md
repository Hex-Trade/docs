# Hextrade Docs (Mintlify)

User documentation for **Hextrade** — how to use the platform. Content is written for people using the dashboard (accounts, algos, copy trade, trading, etc.), not for developers.

## Structure

- **Getting started** — Welcome, first steps, Discord
- **Dashboard** — Accounts, Algos, Copy Trade, Analytics, Economic calendar, Subscriptions
- **Trading** — Trading page, Chart settings
- **Account** — Settings

## Using with Mintlify

- **Repo connected to Mintlify:** Set docs path to `docs`; root config is `docs.json`. Push changes to trigger a build.
- **Editing in Mintlify dashboard:** Copy each `.mdx` into the matching page in the [Mintlify editor](https://dashboard.mintlify.com/hextrade/hextrade/editor/main?file=index.mdx). Match navigation to the page paths in `docs.json`.
